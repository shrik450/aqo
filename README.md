## Introduction

This project is managed with [Poetry](https://python-poetry.org/). To install
dependencies, run:

```bash
poetry install
```

Then, to run the project, use:

```bash
poetry run aqo
```

As it will set up and manage a virtual environment for you. To add dependencies,
use:

```bash
poetry add <package>
```
